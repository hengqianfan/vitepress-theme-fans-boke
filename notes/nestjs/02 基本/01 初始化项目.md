## 安装

- 安装脚手架

::: code-group

```shell [npm]
npm i -g @nestjs/cli
```

```shell [pnpm]
pnpm i -g @nestjs/cli
```

:::



## 创建项目

- 请使用自定义自己的项目名称

```shell
nest new [project-name]
```
```shell
nest new MyNestProject
```
一般是搭建后端，我会这样

```shell
nest new server
```

- 会让你选择自己的包管理器（不懂就用默认）
